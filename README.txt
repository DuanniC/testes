COMO USAR

1. Não abra o index.html pelo file:/// no navegador.
2. Suba a pasta inteira no GitHub.
3. Importe o repositório na Vercel.
4. O site vai usar /api/books automaticamente.

Estrutura:
- index.html
- api/books.js
- package.json

A busca agora passa pelo backend, evitando os erros 403/429 no navegador.
